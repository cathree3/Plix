基于IBM Plex Sans汉字系列修改，先用SC铺底，盖上TC，再盖上JP。

可以显示不规范字形的简体中文。字体文件中.7z压缩包内，每包四款字体，共八款。

工具：中文字體簡繁處理工具 https://github.com/GuiWonder/TCFontCreator
